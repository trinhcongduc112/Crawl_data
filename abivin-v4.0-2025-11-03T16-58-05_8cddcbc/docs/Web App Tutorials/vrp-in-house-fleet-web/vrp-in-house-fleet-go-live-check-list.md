## Web App Checklist

## Web App General Requirement

* Computer requirement: A computer with the following specifications is recommended: CPU Intel core i3; at least 4GB RAM; and of course is equipped with a stable internet connection
* Web browser: Always use [**Google Chrome**](https://www.google.com/chrome/) to access the Web App. The Web app is not fully supported on Mozilla Firefox, Apple Safari or Internet Explorer and it's likely you will run into some UI glitches on those browsers

## User General Requirement

* In order to make yourself fully familiar with the system, it is highly recommended of you to go through the articles below
* [**User Account Setup**](https://docs.abivin.com/docs/web-app-account): This article will help you set up your personal account and provides instructions for such as changing password, reset password
* [**Abivin vRoute System Components**](https://docs.abivin.com/docs/abivin-vroute-components): This article introduces you to all components that are compulsory to make the system usable
* [**Abivin vRoute Glossary**](https://docs.abivin.com/docs/abivin-vroute-glossary). This article outlines the logistics terms that are frequently used within the system
* [**CRUD functions**](https://docs.abivin.com/docs/crud): This article thoroughly describes the indispensable functions that you will use almost everyday - Create, Read, Update and Delete objects
* [**Search & Filter**](https://docs.abivin.com/docs/search-and-filter-functions); [**Page Display & Navigation**](https://docs.abivin.com/docs/page-display-and-navigation-functions) and [**Find Coordinates of Locations**](https://docs.abivin.com/docs/how-to-get-coordinates-of-places): These articles list some handy functions during your use of the Web app

## Fundamental components set up

* At bare minimum, you need to set up the below components in order to have a fully functional flow
* [**Select Management Model**](https://docs.abivin.com/docs/vrp-in-house-fleet-select-model): Select the model after setting up your account 
* [**Manage Organizations**](https://docs.abivin.com/docs/vrp-in-house-fleet-manage-organizations): Setup the Organizations
* [**Manage Users**](https://docs.abivin.com/docs/vrp-in-house-fleet-manage-users): Setup User groups and Users
* [**Manage Customers**](https://docs.abivin.com/docs/vrp-in-house-fleet-manage-customers): Setup Customers
* [**Manage Vehicles**](https://docs.abivin.com/docs/vrp-in-house-fleet-manage-vehicles): Setup Vehicles
* [**Manage Product Categories**](https://docs.abivin.com/docs/vrp-in-house-fleet-manage-product-categories); [**Manage Product**](https://docs.abivin.com/docs/vrp-in-house-fleet-manage-products) and [**Manage Product Inventory**](https://docs.abivin.com/docs/vrp-in-house-fleet-manage-product-inventory): Setup Product Categories; Products and Product Inventory
* [**Manage Actions**](https://docs.abivin.com/docs/vrp-in-house-fleet-manage-actions): Setup Mobile app Actions
* [**Manage Sales Orders**](https://docs.abivin.com/docs/vrp-in-house-fleet-manage-sales-orders): Manage Sales Orders

## Generate and Track Routing Plan

* [**Route Optimization**](https://docs.abivin.com/docs/vrp-in-house-fleet-route-optimization): Create routing plans and generate deliver tasks for the vehicle fleet
* [**Adjust Routing Plan**](https://docs.abivin.com/docs/vrp-in-house-fleet-adjust-routing-plan): Make changes to the routing plan
* [**Missing Orders**](https://docs.abivin.com/docs/vrp-in-house-fleet-missing-orders): Identify why certain orders can not be optimized and how to fix that issue
* [**Delivery Progress Tracking**](https://docs.abivin.com/docs/vrp-in-house-fleet-delivery-progress-tracking): Track the delivery progress in real time
* [**Reports**](https://docs.abivin.com/docs/vrp-in-house-fleet-reports) and [**Dashboard**](https://docs.abivin.com/docs/vrp-in-house-fleet-dashboard): Export data over a period of time or present data visually

## Algorithm configurations

* Below are the algorithm configurations that you can enable to make the routing plan more efficient
* [**Planned Loading/Unloading time**](https://docs.abivin.com/docs/vrp-in-house-fleet-planned-loading-and-unloading-time-calculation): Setup the product loading time at the manufacturing warehouse and the unloading time at the customer location for vehicles
* [**Geographic Clustering**](https://docs.abivin.com/docs/vrp-in-house-fleet-geographic-clustering): Allocate the delivery fleet by geographic clusters
* [**Familiarity Constraint**](https://docs.abivin.com/docs/vrp-in-house-fleet-familiarity-constraint): Setup Customer - Driver familiarity
* [**Site-Dependent Constraint**](https://docs.abivin.com/docs/vrp-in-house-fleet-site-dependent-constraint): Setup Customer - Vehicle types compatibility
* [**Road Constraints**](https://docs.abivin.com/docs/vrp-in-house-fleet-road-constraints): Setup road restrictions by certain parameters
* [**Hard Time Window Constraint**](https://docs.abivin.com/docs/vrp-in-house-fleet-hard-time-window-constraint): Schedule routing plan according to given time windows requested by the customers
* [**Vehicle Lunch Break**](https://docs.abivin.com/docs/vrp-in-house-fleet-vehicle-lunch-break): Reschedule the routing plan according to lunch break time of the drivers

## Other useful functions/features

* [**Salesmen**](https://docs.abivin.com/docs/vrp-in-house-fleet-salesmen-web): Setup the sales department
* [**Cold Chain**](https://docs.abivin.com/docs/vrp-in-house-fleet-cold-chain): Allocate the delivery fleet to meet the temperate based storage conditions of the products

## Mobile App Check List

* Mobile phone requirement: To run Abivin vRoute Mobile app smoothly, we recommend users to use devices with these minimum specifications:
* iPhone: iPhone 5; iOS 6; RAM: 1GB; ROM:16GB
* Android: Android version 6.0; RAM: 1.5GB; ROM: 16GB
* Each mobile phone must at least have 3G internet connection. Enabling high accuracy GPS mode is strongly required. If you use Android phone, follow [**this instruction**](https://support.google.com/maps/answer/2839911?co=GENIE.Platform%3DAndroid\&hl=en) to turn this mode on. If you use iPhone, follow [**this instruction**](https://support.apple.com/en-us/HT203033)
* Read the [**General Instruction**](https://docs.abivin.com/docs/mobile-app-general-instruction) article to get yourself familiar with the basic functions of the Mobile app
* If you are a driver (Delivery man), please read this article to know how to perform your tasks on the Mobile app: [**Driver (Delivery Men)**](https://docs.abivin.com/docs/vrp-in-house-fleet-driver-mobile-app). If you are a Salesman, read this article instead: [**Salesmen**](https://docs.abivin.com/docs/vrp-in-house-fleet-salesman-mobile)
