## Locate Task list

* Navigate to **Tasks > Task List** tab
* This tab lists all tasks that the market researchers will perform on their Mobile apps

## Create Task

* To create a task, follow the steps below
* Hover your mouse over the icon :fa-plus-circle:, then click on **Create Task** icon :fa-pencil:

<Image title="2019-10-11 14_04_07-Window.png" alt={1675} border={true} src="https://files.readme.io/da52d0e-2019-10-11_14_04_07-Window.png">
  Image 1: Create a task
</Image>

* You will be directed to the **Tasks action information** screen
* Input the task information into the fields
* Click **Save** to confirm creating the task
