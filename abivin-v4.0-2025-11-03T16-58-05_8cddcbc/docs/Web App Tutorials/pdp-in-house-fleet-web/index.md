* In this model, the delivery fleet belongs to your organization. When assigned delivery tasks, they will start at the parking garage, travel to one or more locations (Defined as **Origin customers**) to pick up products, then deliver to final destinations (Defined as **Destination customers**) to deliver products
* **Note**: This model can be set up on the same account that's currently running VRP/DC Model. You would need to create separate Branches to run it

> * Apart from VRP/DC model, none of the remaining models can run simultaneously with this model on the same account
