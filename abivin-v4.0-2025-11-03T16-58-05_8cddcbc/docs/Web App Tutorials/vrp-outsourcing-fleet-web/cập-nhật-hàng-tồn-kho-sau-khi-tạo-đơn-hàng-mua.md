* Kiểm tra hàng tồn kho trước khi tạo đơn hàng mua của sản phẩm PRO-001\
  Please checking the inventory before create a purchase order of the PRO-001 product

![1366](https://files.readme.io/6203c22-tk.png "tk.png")

* After creating the purchase order with the number of cases is 200, in the On-POrder column will add to 200, the number of the On-hand column not change

![1366](https://files.readme.io/05f28b3-tk1.png "tk1.png")

* After creating the purchase order, update status of the order to picked & packed, then number of the On-POrder column is reduced, numner of the on-hand is increased 

![1366](https://files.readme.io/9790603-11.png "11.png")

* After creating the purchase order, update a status of the order to canceled, the number of the product    is reduced to 0 and the number of the on-hand column is not change 

![1366](https://files.readme.io/e0301e9-13.png "13.png")
