To check the PSKU you have imported correctly, follow the steps below:

1. Login with the user account belongs to the organization to check
2. Go to the [SKU List] tab, click [Import inventory] button and choose the correct organization to download the sample inventory

<Image title="F4IzC8WsRd_Tmo7HB8oQaQ.png" alt={1368} src="https://files.readme.io/02e04b4-F4IzC8WsRd_Tmo7HB8oQaQ.png">
  Image 1: Download inventory for organization
</Image>

3. Then open the file downloaded in step 2, and open the file you need to check the SKU
4. In your check file, filter the SKU code column that you have imported into the system

<Image title="Untitled.png" alt={1046} src="https://files.readme.io/995ad38-Untitled.png">
  Image 2: filter the SKU code column
</Image>

5. Copy the SKU code in the downloaded file and paste it at the bottom of the SKU code at step 4

<Image title="Untitled.png" alt={1215} src="https://files.readme.io/3c64b0b-Untitled.png">
  Image 3: Copy the SKU code to your file need check
</Image>

6. Then select: Conditional Fomatting => Highlights Cells Rules => Duplicate values\
   If the SKU code is red, then you have imported correctly, if there is any white code is missing or wrong start date and end date.

<Image title="inventory.png" alt={450} src="https://files.readme.io/f4990c2-inventory.png">
  Image 4: Format to compare
</Image>

<Image title="Untitled.png" alt={1158} src="https://files.readme.io/1e273e6-Untitled.png">
  Image 5: Check SKU
</Image>
