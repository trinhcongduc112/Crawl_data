Step 1: Go to the [Organizations] tab, select an organization with type Manufacture then click on the Edit button

![1347](https://files.readme.io/5bcaf3f-N1.png "N1.png")

Step 2: Check into box "Allow Re-delivery Order", then click the ave] b button

![1350](https://files.readme.io/2183da3-N2.png "N2.png")

Step 3: Go to the ers] tab tab,  hover  mouse over on the "+" icon, click on the Fetch old order button

![1362](https://files.readme.io/221ab6c-N3.png "N3.png")

Step 4: Fill information into the form:

* Branch: Select the branch that you want to fetch old order
* Delivery Date: Delivery date the failed orders
* Failed order date: The date that you fetch failed orders\
  Then click on the "Get failed orders" button

![1361](https://files.readme.io/daec53d-N4.png "N4.png")

As you fetch old order with the reason " Not completed" that you submitted on the mobile app, it'll be show on the orders list , mark as green color.

![1364](https://files.readme.io/042052f-N5.png "N5.png")
