## Create single order using Web form

* Please refer to the [**CRUD functions**](https://docs.abivin.com/docs/crud#section-option-1-create-single-objects-using-abivin-vroute-web-form) article to know the general steps about creating single object using web form

<Image title="2019-10-28 16_15_57-Window.png" alt={1905} className="border" border={true} src="https://files.readme.io/543546e-2019-10-28_16_15_57-Window.png" />

* When using Web form, the information must be input exactly in the following sequence:
* **1. Order Code:**
* The Order code is automatically generated in the following format:\
  ***SO-yymmdd-xxxxx***
* In which:
* ***SO*** stands for ***Sales Order***
* ***yy*** is the last two digits of the current year. For example: The current year is ***2019***, then the ***yy*** value would be ***19***
* ***mm*** is the current month. For example: The current month is ***October***, then the ***mm*** value would be ***10***
* ***dd*** is the current date. For example: The current date is ***Sixteenth***, then the ***dd*** value would be ***16***
* ***xxxxx*** is the numerical order of the order. This numerical order will automatically increase each time you open the **Create New Order** screen. For example: ***00001; 00023; 11111*** and so on
* You can also change the order code to your own format. The order code must not contain spaces

<Image title="2019-10-28 16_16_42-Window.png" alt={1908} className="border" border={true} src="https://files.readme.io/00ce674-2019-10-28_16_16_42-Window.png" />

* **2. Customer Code:**
* Click on the field, type the Organization Code/ Organization Name of the customer into the search bar, then select from the drop down menu

<Image title="2019-10-28 16_18_03-Window.png" alt={1908} className="border" border={true} src="https://files.readme.io/a2d3803-2019-10-28_16_18_03-Window.png" />

<Image title="2019-10-28 16_34_07-Window.png" alt={1909} className="border" border={true} src="https://files.readme.io/52b4f62-2019-10-28_16_34_07-Window.png" />

* **3. Product:**
* Click on this field, type the Product Name/Product Code into the search bar. Select from the drop down menu, then click on the **Add** button to add that product into the order
* You can select multiple products by repeating these steps
* The **Expired date** and **Lot Number** fields are optional. If you don't have information of these fields, just leave them blank

<Image title="2019-10-28 16_42_56-Window.png" alt={1909} className="border" border={true} src="https://files.readme.io/2b85a38-2019-10-28_16_42_56-Window.png" />

* Next, you have to input the number of cases and/or number of single items into the **Number Of Cases** and **Number Of Items** fields
* **Note:** If there is zero single item of a specific product in the order being created, you must input ***0*** into the **Number Of Items** field of that product. Similarly, If there is zero case of a specific product in the order being created, you must input ***0*** into the **Number Of Cases** field
* If you accidentally added the wrong product, click on the **Remove** :fa-remove: icon of that product to remove it from the order
* The **Expired date** and **Lot Number** fields are optional. If you don't have information of these fields, just leave them blank

<Image title="2019-10-28 16_59_02-Window.png" alt={1711} className="border" border={true} src="https://files.readme.io/9870fea-2019-10-28_16_59_02-Window.png" />

* **4. Total Price:**
* The **Total Price** field will automatically update as you input the products and the respective number of cases/number of items
* If you want to manually input the order price, click on **Manual Total Price** check box, then input the total price into the **Total Price** field

<Image title="2019-10-28 17_00_25-Window.png" alt={1903} className="border" border={true} src="https://files.readme.io/153babf-2019-10-28_17_00_25-Window.png" />

* **5. Order Date:**
* Click on the this field, then select the date from the drop down menu

<Image title="2019-10-28 17_12_22-Window.png" alt={1909} className="border" border={true} src="https://files.readme.io/708e0ae-2019-10-28_17_12_22-Window.png" />

<Image title="2019-10-28 17_11_42-Window.png" alt={1909} className="border" border={true} src="https://files.readme.io/1c7f6f0-2019-10-28_17_11_42-Window.png" />

* **6. Time Window:**
* Input the Time window exactly in this format: ***hh:mm-hh:mm***
* This field is optional. If there is no information, just leave this field blank

<Image title="2019-10-28 17_07_0s7-Window.png" alt={1909} className="border" border={true} src="https://files.readme.io/b5074cf-2019-10-28_17_07_0s7-Window.png" />

* **7. Split Order; Pickup Order:**
* Specify whether the order being created is a Split Order or a Pickup Order by clicking on the corresponding check box

<Image title="2019-10-28 17_19_51-Window.png" alt={1908} className="border" border={true} src="https://files.readme.io/9bd1da9-2019-10-28_17_19_51-Window.png" />

* **8. Other optional information fields (Invoice Code; Check By):**
* These information fields are optional. If you don't have information, just leave these fields blank

<Image title="20129-10-28 17_19_51-Window.png" alt={1908} className="border" border={true} src="https://files.readme.io/0620e21-20129-10-28_17_19_51-Window.png" />

* After inputting all necessary information, click on **Create** to finish creating the sales order

## Create multiple orders using Excel template

* Please refer to the [**CRUD**](https://docs.abivin.com/docs/crud#section-option-2-create-multiple-objects-using-excel-templates) article to know the general steps about creating multiple objects using Excel template
* Below is the instruction to input in the information fields
* **1. Order Date:**
* Input in either ***dd/mm/yyyy*** or ***mm/dd/yyyy*** format, based on the current Date format of your computer

<Image title="2019-10-29 10_28_30-.png" alt={1906} className="border" border={true} src="https://files.readme.io/85866aa-2019-10-29_10_28_30-.png" />

* **2. Order Code:** 
* The Order code must not contain spaces

<Image title="2019-10-29 10_27_39-Window.png" alt={133} className="border" border={true} src="https://files.readme.io/93afcef-2019-10-29_10_27_39-Window.png" />

* **3. Order Type:**
* Always input this value: **SALES**

<Image title="2019-10-29 10_26_59-Window.png" alt={113} className="border" border={true} src="https://files.readme.io/64b1dde-2019-10-29_10_26_59-Window.png" />

* **4. Partner Code:**
* Copy the **Customer Code** of the customer on Web app, then paste into this cell
* The Customer Code can be found under **Customer Code** column in **Partners > Customer List** tab

<Image title="2019-10-29 08_45_57-Window.png" alt={1905} className="border" border={true} src="https://files.readme.io/a5f41a4-2019-10-29_08_45_57-Window.png" />

* **5. Product Code:**
* Copy the **Product Code** of the product on Web app, then paste into this cell
* The Product Code can be found under **Product Code** column in **Products > Inventory** tab

<Image title="2019-10-29 08_51_23-Window.png" alt={1905} className="border" border={true} src="https://files.readme.io/93cf25d-2019-10-29_08_51_23-Window.png" />

* If there are multiple product loaded in one order, each product will be placed on one dedicated row

<Image title="2019-10-29 08_52_34-Window.png" alt={564} className="border" border={true} src="https://files.readme.io/b1df8d6-2019-10-29_08_52_34-Window.png" />

* **6. Time Window:**
* If there is a time window, always input in the following format: ***hh:mm-hh:mm*** (24 hour format)
* The time point must not exceed ***23:59***
* If there is no time window, leave the cell blank

<Image title="2019-10-29 08_58_21-Window.png" alt={135} className="border" border={true} src="https://files.readme.io/b68ac2a-2019-10-29_08_58_21-Window.png" />

* **7. Number of Cases; Number of Items:**
* Input the quantity of cases and single items of the product into the **Number of Cases** and **Number of Items** cells
* If there is zero case being loaded, input ***0*** into the **Number of Cases** cell. Do not leave that cell blank
* If there is zero single item being loaded, input ***0*** into the **Number of Items** cell. Do not leave that cell blank
* If there are multiple products, input the quantity of cases/single items of each product into its respective row

<Image title="2019-10-29 08_59_59-Window.png" alt={563} className="border" border={true} src="https://files.readme.io/71f9cea-2019-10-29_08_59_59-Window.png" />

* **8. Total Price:**
* If you want the system to automatically calculate the order price, leave this cell blank

<Image title="2019-10-29 09_56_33-Window.png" alt={1899} className="border" border={true} src="https://files.readme.io/54fd91c-2019-10-29_09_56_33-Window.png" />

* If you don't want the system to automatically calculate the price, input your own price into the cell. If the order contains multiple products, input the price of each product into its respective cell. The values will be accummulated after uploading the Excel template onto Web app

<Image title="2019-10-29 09_26_40-Window.png" alt={1899} className="border" border={true} src="https://files.readme.io/357c971-2019-10-29_09_26_40-Window.png" />

* **9. Customer Discount; Sale Discount; Promotion Discount; IMVD Discount:**
* **Note:** These discounts will only take effect if you let the system automatically calculate the total price of the order
* The IMVD discount will be subtracted directly from the price of each product (See blue rectangles)
* The Customer Discount; Sale Discount; Promotion Discount must be input the same for every products of an order (See red rectangles)
* The total price will be calculated in this formula: ***Sum of product prices (After subtracting the IMVD discounts) - (Sum of Customer Discount; Sale Discount & Promotion Discount) = Total Order Price*** (See green rectangle)

<Image title="2019-10-29 09_46_53-Window.png" alt={1899} className="border" border={true} src="https://files.readme.io/b05a2b5-2019-10-29_09_46_53-Window.png" />

* **10. Pickup Order:**
* If the order being created is a Pickup Order, input ***1*** into the cell
* If the order being created is not a Pickup Order, input ***0*** into the cell, or leave the cell blank

<Image title="2019-10-29 10_03_04-Window.png" alt={134} className="border" border={true} src="https://files.readme.io/2786d04-2019-10-29_10_03_04-Window.png" />

* **11. Splitted Order:**
* If the order being created is a Split Order, input ***TRUE*** into the cell
* If the order being created is not a Split Order, input ***FALSE*** into the cell, or leave the cell blank

<Image title="2019-10-29 10_05_26-Window.png" alt={136} className="border" border={true} src="https://files.readme.io/6134c59-2019-10-29_10_05_26-Window.png" />

* **12. Service Time (minutes):**
* Leave this cell blank
* **13. Lot Number; Expired Date:**
* These fields are optional. Leave them blank if you don't have information
* If the product has an Expired Date, input in either ***dd/mm/yyyy*** or ***mm/dd/yyyy*** format, based on the current Date format of your computer

## Order color highlights

* On **Orders > Sales Orders** tab, the order will be highlighted with some background colors to indicate its status
* **No highlight:** Normal order (Both locked or unlocked)

<Image title="2019-10-29 16_40_15-Window.png" alt={1654} className="border" border={true} src="https://files.readme.io/bb0044f-2019-10-29_16_40_15-Window.png" />

* **Red color highlighted:** The Customer of the order has no coordinate information

<Image title="2019-10-29 15_53_34-Window.png" alt={1710} className="border" border={true} src="https://files.readme.io/3f6d64a-2019-10-29_15_53_34-Window.png" />

* **Yellow color highlighted:** There are two cases
