In Abivin vRoute database, there are 3 types of Users:

* Owner user: The first user, created before having any organization. This user has full access to everything it created.
* Admin user: Users that belong to Admin group are called admin users. They are equivalent to the Director/Manager in businesses.
* Normal user: All other users, who are not Owner or Admin. They are equivalent to ordinary employees who are specialized in certain tasks.
