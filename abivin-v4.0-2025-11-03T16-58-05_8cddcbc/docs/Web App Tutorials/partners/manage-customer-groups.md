## Import the customer groups

Step 1: Download the data sample from the system

![1366](https://files.readme.io/c216701-N1.png "N1.png")

Step 2: Fill the information of the customer group into the excel file

![1358](https://files.readme.io/7f1fed2-N2.png "N2.png")

Case 1: if the Associated Group Code is blank, the shipment to customers in this Group will be optimized and assign to drivers normally.\
Case 2: If it's filled with its own code (or we call it self-associated code), the delivery plan will only include orders to Customers in this Group.\
Case 3: If it's filled with associated group code, the delivery plan will include Customers in the associated Groups.\
Step 3: Import the customer group file by select the file.
