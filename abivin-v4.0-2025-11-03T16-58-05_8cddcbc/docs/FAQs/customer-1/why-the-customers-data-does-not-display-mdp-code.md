## Why the Customers' data does not display the MDP code?

**:fa-angle-right:Case:** After uploading the Customers' file on the Website, you randomly check some customers information and realize that the MDP code is displayed under the format of ""\[object object] instead of the right code stated in the Excel file.

![1920](https://files.readme.io/e53b368-customer1.png "customer1.png")

**:fa-angle-right:Reason:** The MDP code stated in the Excel file is written under the wrong format which is foumla.

![1920](https://files.readme.io/d0b2ab1-customer2.png "customer2.png")

> ðŸ“˜ Notice
>
> The right format of a MDP code is number and/or text value with no spaces.

**:fa-angle-right:Solution:** Open your Excel file and change the MDP field format to the required one. Then re-upload the Customers' file and the new data will automatically overwrite the recent one.
