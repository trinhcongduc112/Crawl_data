Below are some common circumstances explained why you cannot upload the Order file to the Website.

## 1. Duplication Order Code.

* After uploading the Order file, the system displays a notification of nvalid Upload Data] s stated that Order Code is duplicated. 

![1920](https://files.readme.io/653efe9-Order_1.png "Order 1.png")

:fa-angle-right: Solution: Go back to your Order file, modify the duplicated order and re-upload the file to the Website.

![1920](https://files.readme.io/0acab37-order_2.png "order 2.png")

## 2. Duplication Order Code with different Partner Code.

* After uploading the Order file, the system displays a notification of  Upload Data] stated  stated that Order Code is duplicated with different Partner Code. 

![1920](https://files.readme.io/89ddbe2-order_3.png "order 3.png")

:fa-angle-right: Solution: Go back to your Order file, modify the duplicated orders and re-upload the file to the Website.

![1886](https://files.readme.io/b20722d-order_4.png "order 4.png")

## 3. Order has Non-Existed Product Code.

* After uploading the Order file, the system displays a notification of d Data] stated Non-Ex stated Non-Existed Product Codes.

![1920](https://files.readme.io/21f45af-order_5.png "order 5.png")

* Go to Products and Check the Product Codes by comparing Product Codes in the Website with those in your file.

![1920](https://files.readme.io/0e4af40-order_6.png "order 6.png")

:fa-angle-right: Solution: Modify the Product Codes column in your file by adding existed products or deleting non-existed products. Then re-upload the Order file to the Website.

![1919](https://files.readme.io/38b60c5-order_7.png "order 7.png")
