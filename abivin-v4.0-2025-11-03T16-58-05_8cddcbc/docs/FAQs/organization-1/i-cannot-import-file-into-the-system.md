## 1. Check your files.

* Does your file contain more than 1 sheet?

![2378](https://files.readme.io/81d2c54-3lSFHc9W6o1jpkXFLPcyslBCW9fyKMwaiQ.png "3lSFHc9W6o1jpkXFLPcyslBCW9fyKMwaiQ.png")

:fa-angle-right: The system allows to import only 1 sheet. Please delete other sheets and re-upload the file into the system.

## 2. Check your data.

* Does your file contain link?

![1988](https://files.readme.io/6460664--pi37yacqjDRaPPFj6CwfFVomvoxpnjLIg.png "-pi37yacqjDRaPPFj6CwfFVomvoxpnjLIg.png")

:fa-angle-right: Please click the right mouse button and choose Remove Hyperlinks, then re-upload the files into the system.

## 3. Wrong format when upload files?

:fa-angle-right: You should download the files' template, then copy the data into the corresponding column before uploading into the system.
