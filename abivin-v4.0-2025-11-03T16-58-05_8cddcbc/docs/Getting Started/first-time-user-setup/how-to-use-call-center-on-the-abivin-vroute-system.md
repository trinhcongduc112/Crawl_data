A call center is an open work area for agents. Call Agents are customer service workers who receive calls or make calls to answer customer inquiries.\
To set up a call for every Call Agents in the system, follow the 3 steps below:\
Step 1: Log into the system, go to the [Organizations] tab and click the [User List] tab

* When a call comes from the customer, the call agent will press the blue button to receive the incoming call

- If there are two call agents available on the system that have two incoming calls, the two calls will be divided between two call agents.
- In case there are two or more call agents available on the system, if a call agent is receiving the call but there is another coming call, the incoming call will be directed to the other call agent.
- If there is an incoming call while there is only one call agent on the system, who is also answering a call, the current call will be paused, and the incoming call will be prioritized.
- In case there is one call agent who is receiving the call, one other call agent who is available on the system that has a call will have a bell to notify that person.
- If there is an incoming call that the call agent refuses, press the red  button  button to cancel the incoming call.
