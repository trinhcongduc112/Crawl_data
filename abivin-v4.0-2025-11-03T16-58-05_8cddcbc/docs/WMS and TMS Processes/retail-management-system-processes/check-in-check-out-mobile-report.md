## Create Check-in/Check-out Mobile task

Admin account creates check-in/check-out task via Abivin vRoute Form Builder. 

Retail representative will be able to find these tasks once they log in mobile. The user interface will be similar to this:

<Image title="Screenshot_20190531-120458_Abivin vRoute v30.jpg" alt={1080} className="border" border={true} src="https://files.readme.io/20deade-Screenshot_20190531-120458_Abivin_vRoute_v30.jpg" />

Retail representative can take photos corresponding to each task (Maximum of 3 photos per task) and tap on "Submit" to submit the task to Abivin vRoute server.

<Image title="Geometri photo app.gif" alt={320} className="border" border={true} src="https://files.readme.io/aa015c9-Geometri_photo_app.gif" />

## Export Monthly Check In/Check Out report

Admin account can then export the monthly Check In/Check Out report.

The Check In/Check Out report will have the following information: Account, Store Code, Store Name, Done? Staff Code, Staff Name, Time In, Time Out, Actual Check-in Latitude, Actual Check-in Longitude, Date.
