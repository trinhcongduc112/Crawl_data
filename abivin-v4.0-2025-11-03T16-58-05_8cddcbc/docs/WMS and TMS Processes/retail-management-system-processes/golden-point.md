## PSKU Distribution

The PSKU distribution report retrieves data from the SOOS(Share out of Stock) task.\
After users submit Task SOOS on app mobile, admin export report PSKU Distribution on web.

![1362](https://files.readme.io/4eabcc1-8.png "8.png")

## Visibility

* First, Admin creates task Visibility for users. Depending on the user of which branch the admin creates the corresponding task: CCC visibility, HSM visibility, CVS visibility, MM visibility, IS visibility.
* In task visibility, the questions will be different for each branch.
* After task created, user login on mobile app will be display as follow:

![540](https://files.readme.io/ab3dcc0-Screenshot_2018-07-31-16-47-55.png "Screenshot_2018-07-31-16-47-55.png")

* User choose yes/no for question and choose from library or take photo. Then, click submit.

![540](https://files.readme.io/4993600-Screenshot_2018-07-31-16-53-41.png "Screenshot_2018-07-31-16-53-41.png")

## Initiative

**Upload Tagets**\
Download the file  sample Targets : 

1. Select Settings
2. Select tab Targets.
3. Click Import
4. Click Dowload Sample 

![1359](https://files.readme.io/e0bdc70-1.png "1.png")

![1363](https://files.readme.io/7522cdd-2.png "2.png")

Enter information in import target file:\
Open the download file, enter the information into the file under the instructions below

1. Enter Targets Code (users are free to set)
2. Enter Target Type : PROMOTION.
3. Enter Title Target.
4. Enter Start Date of Target.
5. Enter Due Date of Target.
6. Enter SKU of Target

![1919](https://files.readme.io/551c064-3.png "3.png")

Next, import the target file 

1. Click on " Drop import file here or click to upload"
2. Choose file from computer.

![1919](https://files.readme.io/ca3d5fc-4.png "4.png")

![1919](https://files.readme.io/a263ba8-5.png "5.png")

After succesfully uploading the target file, the list of targets will be displayed as follows:

![1364](https://files.readme.io/39b764b-6.png "6.png")

Task initiative shown on the app as shown below:

![540](https://files.readme.io/3992c65-Screenshot_2018-07-31-10-32-03.png "Screenshot_2018-07-31-10-32-03.png")

User choose Yes/No and upload image ( if any). Then, click submit.

![540](https://files.readme.io/271d1c9-Screenshot_2018-07-31-14-47-01.png "Screenshot_2018-07-31-14-47-01.png")

Finally, Admin will export report intiative on web when user completed task initiative.
