## How to create a promotion program

The steps create a promotion program:\
 The first, the user download template file of promotion program and fill data to file.\
Data entered into the file is guided below:

* Promotion code : Code of promotion program, data is unique and required.
* Organization Code : Code of organization
* PostmailID
* Brand 
* Size  
* Mechanic Description 
* Start Date : Start Date of promotion program.
* Due Date : Due date of  promotion program.
* Duration : number of days allowed to take photo.
* Action code : Enter "PHOTO-APP".

![1907](https://files.readme.io/4af21b2-2.png "2.png")

After preparing the import file, the user follows these steps : 

1. Click Settings.
2. Choose Promotion program.
3. Hover over the + sign , click Import
4. Choose file upload.\

## How to delete promotion program

To remove a promotion, the user follows these steps : 

1. Choose the promotion program to delete.
2. Click Delete
