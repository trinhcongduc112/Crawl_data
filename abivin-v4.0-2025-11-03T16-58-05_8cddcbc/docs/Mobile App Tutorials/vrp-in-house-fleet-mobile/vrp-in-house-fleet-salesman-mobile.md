## View Order details

* To view all sales orders delivered to the customers associated with you, you can follow the steps below:
* Tap on **Orders** tab
* All sales orders delivered to the customers with whom you associate will be listed on **Sales Orders** sub tab
* To view the details of an order, you can simply tap on the corresponding order code

![500](https://files.readme.io/fd41d53-View_Orders.gif "View Orders.gif")

* To search for a specific order, you can tap on the icon :fa-search: on the top toolbar, then input the order code into the search bar. The app will filter and return the order which has that order code

* If you have been assigned the permission to create sales orders on behalf of the customers which you manage, you can do so by following the below steps:
* Tap on **Orders** tab
* On this tab, tap on the orange :fa-plus-circle: icon, then tap on **New Order** icon :fa-pencil:

![416](https://files.readme.io/0ecc81d-New_Order_button.png "New Order button.png")

* You will be directed to the **Create Order** screen

![416](https://files.readme.io/3f87764-Create_Order_Screen.png "Create Order Screen.png")

### Input general order information

* Tap on the text **Select** below the text **Customer**, you will be navigated to the **Select Customer** screen, where you can choose the customer

![416](https://files.readme.io/8ece9a3-Select_Customer.png "Select Customer.png")

* You can swipe the customer list up and down to find the appropriate customer. You can also tap on the icon :fa-search:, then input the name of the customer into the search bar to quickly filter out the appropriate customer

![416](https://files.readme.io/49237d9-Search_Cus.png "Search Cus.png")

* After selecting the customer, tap on the icon :fa-calendar-o: below the text **Sale order date** to select the date on which the order being created will be performed

* Next, consequently tap on **Start time frame** and **End time frame** to specify the time window on during which the order being created has to be performed on the date selected above

![416](https://files.readme.io/f709d67-Date.png "Date.png")

* If the customer doesn't require a time window, you don't need to input anything into these fields

### Input product information

* Tap on the icon :fa-plus: on the row **Product**

![416](https://files.readme.io/f57cc6c-Product_Button.png "Product Button.png")

* You will be navigated to the **Select Product** screen

![416](https://files.readme.io/ab7ad41-Select_Product.png "Select Product.png")

* You can swipe the product list up and down to find the appropriate product. You can also tap on the icon :fa-search:, then input the name of the product into the search bar to quickly filter out the appropriate product. After finding the correct product, tap on that product to add it to the order

![416](https://files.readme.io/e17b285-Select_Product.png "Select Product.png")

* If you incorrectly input the wrong product, tap on the trash bin icon :fa-trash: to remove the product

![416](https://files.readme.io/43d995b-Trash_Bin.png "Trash Bin.png")

* After selecting the product, you will be redirected back to the **Create Order** screen. Here you will have to specify the quantity of the whole case and/or quantity of single items for the selected product, by tapping on the corresponding numbers next to the texts **Case** and **Item**, input the correct case/single item quantity on the pop-out numeric keypad, then tap on **Done**

![416](https://files.readme.io/2ac846c-20200825_162603.png "20200825_162603.png")

* If you incorrectly input the quantity, you can tap on the plus and minus icons :fa-plus-square-o: and :fa-minus-square-o: to increase/decrease the quantity as you wish. You can also tap directly on the numbers input earlier and input the new numbers

![416](https://files.readme.io/63d6c32-Quantity.png "Quantity.png")

* If there are discount/promotion campaigns applied to the selected customer/product, the detail discount/promotion effect will be listed out under **Discount** and **Promotions** sections

- The total value of the order will automatically be calculated and displayed on the orange **Total** section at the bottom of the screen

![416](https://files.readme.io/00863bb-Total.png "Total.png")

### Submit order to the order approver

* After inputting all necessary information of the order, if you are sure that the information is correct, you can submit the order to the order approver on Web app by tapping on the text **Create** on top of the screen

![416](https://files.readme.io/ac16c3e-20200825_162926.png "20200825_162926.png")

* The order will immediately be submitted to the order approver

## Edit submitted orders

* After the orders have been submitted, if the order approver has yet to approve/reject the order, then you can still edit those orders (If you have been assigned the permission to edit Orders)
* To edit Orders, follow the steps below:
* Tap on the Order code of the order which you want to edit

- On the **Order Detail** screen, tap on the icon :fa-pencil: on top of the screen. Then you will be able to edit the information of that order

![416](https://files.readme.io/714f9c4-Edit_Order.png "Edit Order.png")

* After making necessary changes, tap on the text **Save** to save the order

![416](https://files.readme.io/1a59e54-Save_Edit.png "Save Edit.png")
