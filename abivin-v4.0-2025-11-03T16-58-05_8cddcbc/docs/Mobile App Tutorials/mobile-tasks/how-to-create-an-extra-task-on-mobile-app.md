## Config an organization to use extra tasks

Step 1: Go to the Organiztion tab, select an organization have Branch type, then click Edit button

![1344](https://files.readme.io/4b9099e-2019-02-26_11-06-11.jpg "2019-02-26_11-06-11.jpg")

Step 2: Click into the box "Allow driver to create Extra Task", then save it

![1899](https://files.readme.io/4407d4e-2019-02-26_11-08-15.jpg "2019-02-26_11-08-15.jpg")

## Submit tasks on mobile

To create an extra tasks, delivery men performs the task of load product, then icon "+" will appear.\
Step 1: Click on the yellow " + " icon at the bottom

![720](https://files.readme.io/c7421bc-a7bf6be7c5ef27b17efe.jpg "a7bf6be7c5ef27b17efe.jpg")

Step 2: Take photo and fill the required information into the form\
