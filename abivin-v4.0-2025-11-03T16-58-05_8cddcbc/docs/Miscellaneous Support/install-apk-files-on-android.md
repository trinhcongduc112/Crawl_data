Step 1: Go to Settings on Android, select Security.

![351](https://files.readme.io/0740ab7-s1.png "s1.png")

Step 2: Scroll down, select Unknown sources

![350](https://files.readme.io/e53b443-s2.png "s2.png")

Step 3: Find the APK file you downloaded, click to install (You can download this file from the link given to you from Google Drive. Please ask Abivin team for the link if you haven't got it).

![347](https://files.readme.io/5d56c2f-s3.png "s3.png")

Step 4: Perform the installation.

![349](https://files.readme.io/f81dfcf-s4.png "s4.png")

After install, click application and login to start

![345](https://files.readme.io/c7caaa2-s5.png "s5.png")
