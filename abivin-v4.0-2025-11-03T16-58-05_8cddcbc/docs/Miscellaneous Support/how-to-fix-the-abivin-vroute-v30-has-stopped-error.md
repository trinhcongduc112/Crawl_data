When a user encounters an "Abivin vRoute v3.0 has stopped" error, follow 3 steps below to fix:\
Step 1: Go to Settings -> Apps

<Image title="saZ7IhHmTVSeyBsN6D2eYA.png" alt={719} src="https://files.readme.io/0fb2f80-saZ7IhHmTVSeyBsN6D2eYA.png">
  Image 1: Go to apps setting
</Image>

Step 2: Find the Abivin vRoute v3.0 application

<Image title="2.png" alt={357} src="https://files.readme.io/36ca161-2.png">
  Image 2: Go to the vAppX apps
</Image>

Step 3: Click clear data and clear cache

<Image title="3.png" alt={422} src="https://files.readme.io/b1ea24f-3.png">
  Step 3: Clear cache and clear data
</Image>

Then restart the apps and sign in again to start using
